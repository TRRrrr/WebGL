Class okSkAnimPlayer

骨骼动画播放器类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okSkAnimPlayer()
okSkAnimPlayer提供播放骨骼动画的统一功能接口封装。开发者使用okSkAnimPlayer可以实现骨骼动画的常用播放、混播等功能，而不必直接操作骨骼动画(okSkAnimation)对象。

Class okFont

字体类，包括一系列字体属性，例如字体家族、字号、字体粗细等等。所有与字体相关的操作接口都封装在此类中。

Method Detail
drawText(targetRect, text, color, renderTarget)
在渲染目标的指定区域使用指定颜色和透明度绘制文本。
Parameters:
{okRect} targetRect
渲染目标中的绘制区域。如果本参数为null，那么就把整个渲染目标当做绘制区域。
{String} text
要绘制的文本。
{okVec4} color
绘制的文本的颜色和透明度，格式为RGBA，其中每个分量的值都处于[0, 1]的区间。
{okTexture} renderTarget
渲染目标。如果本参数为null，那么将会在当前字体的渲染上下文中进行绘制。
enableAutoWrap(bAutoWrap)
设置当调用drawText函数时，是否自动包装文字。默认值是true。
Parameters:
{Boolean} bAutoWrap
设置当调用drawText函数时，是否自动包装文字。
{Boolean} isAutoWrapEnabled()
检测字体是否处于自动包装模式下。
Returns:
{Boolean} 如果该字体对象处于自动包装模式下，则返回true；反之，则返回false。
setCodePageMaxNum(maxNum)
设置文字缓存可以同时储存的字符的最大数量，它关系到字体模块的渲染性能。
Parameters:
{Number} maxNum
文字缓存可以同时储存的字符的最大数量，默认值是5。
setCodePageSize(width, height)
设置文字缓存尺寸。文字缓存的宽度和高度关系到此文字缓存中可以储存的字符的数量。
Parameters:
{Number} width
文字缓存宽度，应当是2的整数次幂；如果不是2的整数次幂，那么函数将会自动增大到下一个最小的2的整数次幂的数值。默认值是512。
{Number} height
文字缓存高度，应当是2的整数次幂；如果不是2的整数次幂，那么函数将会自动增大到下一个最小的2的整数次幂的数值。默认值是512。

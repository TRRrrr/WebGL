Class okFrustum

平截头体类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okFrustum()
平截头体在计算机图形学中通常用于描述屏幕上可见的三维区域，被称为“Viewing Frustum”，也就是相机的可视区域。平截头体是由金字塔形切开形成的；
另外特别需要说的是，平截头体裁剪是用于可视裁剪的重要方法。

指定平截头体的某一面。可选的值是[FTPLANE_LEFT, FTPLANE_RIGHT, FTPLANE_BOTTOM, FTPLANE_TOP, FTPLANE_NEAR, FTPLANE_FAR]。

{okVec3} getPoint(PointIdx)
返回此平截头体的指定端点。
Parameters:
{Number} PointIdx
顶点索引。
0:近面、底面和左侧面的交点; 1:近面、底面和右侧面的交点;
2:近面、顶面和右侧面的交点; 3:近面、顶面和左侧面的交点;

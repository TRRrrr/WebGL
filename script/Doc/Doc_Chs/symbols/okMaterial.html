Class okMaterial

材质类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okMaterial()
okMaterial提供了对渲染材质信息的抽象封装，材质信息定义了一个物体被绘制的方式, 包括光照属性、纹理信息等。

开启或关闭Alpha测试（Alpha Test）。 Alpha测试会将片元的Alpha值与一个常量进行比较，只有符合条件的像素才会通过Alpha测试，继续下一轮测试直至最终绘制；不符合条件的像素，也就是没有通过Alpha测试的像素将会被忽略。

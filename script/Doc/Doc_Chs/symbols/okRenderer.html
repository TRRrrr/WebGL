Class okRenderer

Oak3D Engine渲染器。

Class Summary
Constructor Attributes Constructor Name and Description
<private> <inner>  
okRenderer(engineSetting, engine)
okRenderer为Oak3D Engine的渲染器对象，负责3D内容的实际绘制。开发者不能直接创建渲染器对象，它是okEngine对象的内置的组成部分。

Method Attributes Method Name and Description
 
beginView(Cam, KeepBackGround)
开始一个View的绘制。 在执行绘制过程中，此接口应与endView成对出现，且不可嵌套。任何实际的绘制代码都应属于某一个View。
一个View是由相机的视场定义的，也就是随后所有操作的目标渲染区域。

{Null} beginView(Cam, KeepBackGround)
开始一个View的绘制。 在执行绘制过程中，此接口应与endView成对出现，且不可嵌套。任何实际的绘制代码都应属于某一个View。
一个View是由相机的视场定义的，也就是随后所有操作的目标渲染区域。
Parameters:
{okCamera} Cam
当前View使用的相机对象。
{Boolean} KeepBackGround
指定此View是否保留所属背景区域。默认值是false。所以默认情况下，本方法会清楚所属的背景区域。
Returns:
{Null}

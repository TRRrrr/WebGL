Namespace okFontManager

okFontManager 管理当前系统中的所有字体。

<static>  
okFontManager.deleteFont(rc, fontFamily, fontSize, fontWeight, fontVariant, fontStyle)
根据参数中指定的属性删除一个字体对象。如果符合属性值的字体对象存在，那么它将被删除；如果不存在，则不会发生任何事。本函数也可以只接收一个类型为okFont的参数，此时，本函数将会删除该okFont对象。
注意：如果你引用了指定的okFont对象，那么你必须手动解除引用以释放内存完成垃圾回收。本函数仅仅是从okFontManager中移除该okFont对象，而不是直接删除该okFont对象本身。

<static> {okFont} okFontManager.createFont(rc, fontFamily, fontSize, fontWeight, fontVariant, fontStyle)
创建一个新的字体对象，或者根据指定的属性获取既存的字体对象。
Parameters:
{renderContext} rc
WebGL设备渲染环境
{String} fontFamily
新增字体对象的字体家族名，例如“Courier New”。
{Number} fontSize
新增字体对象的字号。
{String} fontWeight
本参数用于设置显示的文本中的字符的粗细。例如“normal”、“bold”、“bolder”、“lighter”、900等等。
{String} fontVariant
本参数用于指定文本是否要显示为小型大写字母（small-caps）。可选的值包括“small-caps”、“normal”。
{String} fontStyle
本参数用于指定字体样式。可选的值包括“normal”、“italic”、“oblique”。
Returns:
{okFont} 函数将会根据参数中指定的字体属性返回一个okFont对象。如果拥有相同属性的okFont对象已经存在，那么将会返回此既存的okFont对象；否则，将会创建一个新的okFont对象。
<static> okFontManager.deleteFont(rc, fontFamily, fontSize, fontWeight, fontVariant, fontStyle)

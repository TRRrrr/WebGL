Class okSkAnimation

骨骼动画类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okSkAnimation()
骨骼动画指由若干带有父子关联性的骨骼及附加在骨骼上的一系列关键帧组成的用于表达一段动作的对象。骨骼动画是实时3D应用于最常用的动画种类之一，一般用于角色动作和动态物体等.

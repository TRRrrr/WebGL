Class okTexture

纹理

Class Summary
Constructor Attributes Constructor Name and Description
 
okTexture(Context)
okTexture是WebGL标准中纹理对象在Oak3D中的对象封装。纹理通常指3D绘制中引用到的图像数据。可理解为像素的集合, 是3D渲染中最常用到的数据类型之一。Texture也可以作为渲染目标去接收Frame Buffer的输出。

okTexture(Context)
okTexture是WebGL标准中纹理对象在Oak3D中的对象封装。纹理通常指3D绘制中引用到的图像数据。可理解为像素的集合, 是3D渲染中最常用到的数据类型之一。Texture也可以作为渲染目标去接收Frame Buffer的输出。

将此Texture作为WebGL上下文的对应类型，对应激活层的当前Texture。
Parameters:

指定绑定的texture的激活纹理层。默认值是0。
Deprecated:

Parameters:
{Enum} TexType
指定纹理类型，可选的值是[TEXTURE_2D, TEXTURE_CUBE_MAP]。
{Number} Width
指定纹理的宽度，若载入的数据源为image或video，则这里赋-1即可，尺寸将自动由html对象读取。
{Number} Height
指定纹理的高度，若载入的数据源为image或video，则这里赋-1即可，尺寸将自动由html对象读取。。
{Enum} Format

{Enum} DataType

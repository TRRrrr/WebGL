okProgram(Context)
Program Object是WebGL中的Shader绑定容器。开发者可将特定的Vertex Shader与Fragment Shader组合绑定到一个Program中，需要渲染管线使用这个Shader组合时，只需将此Program Object指定为WebGL上下文的当前Program即可。

link(RetErr)
链接当前绑定的Vertex Shader与Fragment Shader，生成有效的Program。 本方法将会对Program对象进行链接操作。
绑定在此Program中的Vertex Shader将用于创建一个可执行程序运营在可编程的顶点处理器上；
绑定在此Program种的Fragment Shader将用于创建一个可执行程序运行在可编程的片元处理器上。
 
releaseProgram()
释放当前WebGL Program Object资源。 本方法将会释放由此Program生成的WebGL资源。在调用本方法后，此Program将不再可用，直至下次重新链接。

指定绑定的Array Buffer Object。
{Number} ElementCount
指定Array Buffer中几个元素对应一个顶点。比如:
若存储<x,y,z>形式的顶点位置数据，则此项应设为3；
若存储<u, v>形式的纹理坐标数据, 则此项应设为2。
{Number} Offset Optional
指定从当前Array Buffer的第几个元素开始。默认值是0。
{Number} Stride Optional
指定相邻两顶点距离的元素数，一般用于Array Buffer存储多类数据的情况。比如:
Array Buffer存储顶点位置与法线，形式为<x, y, z><nx, ny, nz><x, y, z>...；则位置Attribute的stride应为6。本参数默认值是0。

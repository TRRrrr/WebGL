Class okShader

Shader类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okShader(Context, Type)
WebGL支持使用GLSL ES语言对渲染管线中的顶点处理单元（Vertex Processing）与像素处理单元（Fragment Processing）的行为进行编程。Shader Object是WebGL可编程管线的程序容器，通过为Shader指定不同的处理程序可达到自定义渲染行为的目的。
Method Summary
Method Attributes Method Name and Description
 
compile(Def, RetErr)
编译此Shader Object中的源代码，并将宏定义参数列表中的所有宏定义都加入到代码的起始处。注意：Shader Object只有成功编译后，才能使用。 本方法将把源代码编译成为WebGL可接受的二进制码。
 
loadSource(Code, Include)
载入Shader源代码。
 
releaseShader()
释放当前的Shader。 本方法将会释放由此shader产生的WebGL资源。在调用本方法后，此shader将不再可用，直至下次重新编译。
 
releaseSource()
清除此okShader Object已载入的源代码。 本方法将释放此shader种包含的源代码。
 
setFloatPrecision(FloatPrecision)
指定当前Shader编译时使用的浮点精度。
Class Detail
okShader(Context, Type)
WebGL支持使用GLSL ES语言对渲染管线中的顶点处理单元（Vertex Processing）与像素处理单元（Fragment Processing）的行为进行编程。Shader Object是WebGL可编程管线的程序容器，通过为Shader指定不同的处理程序可达到自定义渲染行为的目的。
Parameters:
{WebGLContext} Context
WebGL设备渲染环境。
{Enum} Type
指定Shader的类型。可选的值是[VERTEX_SHADER, FRAGMENT_SHADER]。
Returns:
{okShader} 一个新的okShader对象。

编译此Shader Object中的源代码，并将宏定义参数列表中的所有宏定义都加入到代码的起始处。注意：Shader Object只有成功编译后，才能使用。 本方法将把源代码编译成为WebGL可接受的二进制码。
Parameters:

一个格式为的宏定义列表。

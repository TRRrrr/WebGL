rot(Space, Angle, DirX, DirY, DirZ, SelfAffect)
对当前矩阵做一个绕指定轴的旋转变换。 本方法实际是将当前矩阵与一个旋转矩阵做乘法运算，然后返回结结果。
乘法的顺序是由“Space”参数决定的：SPACE_LOCAL意味着“当前矩阵 x 旋转矩阵”；SPACE_WORLD意味着“旋转矩阵 x 当前矩阵”。

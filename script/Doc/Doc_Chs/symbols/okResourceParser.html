Namespace okResourceParser

所有Oak3D内置的资源解析都由此类来处理。okResourceParser是一个系统常量Object，提供了对Oak3D资源文档的统一读取接口。开发者可使用okResourceParser提供的各种接口直接将外部资源档案的内容解析到Oak3D的对应对象中.

<static> {Boolean} okResourceParser.registerModelParser(DocType, Parser)
注册一个自定义模型文档解析器。 Oak3D 允许开发者开发自定义的模型解析器支持各种不同的模型档案。
Parameters:
{Enum} DocType
档案类型. [RESDOCTYPE_CUSTOM + i (i >= 0)].
{Function} Parser
解析器函数. 解析器函数的原型为: Boolean function(okModel model, String docText, Array meshMask)。 这个函数应负责解析docText内容，并将数据填充到model对象中。 'meshMask'参数用于实现部分加载，如果你的解析器不需要部分加载功能，忽略此参数即可。 解析器的返回值用于表面解析过程是否成功完成。
Returns:
{Boolean} 若注册成功，返回true, 否则，返回false.
See:
okModel

createAttribute(Name, Len, Data, Usage)
创建一个新的顶点属性列表。 okMesh对象中的属性与WebGL中的属性略有不同。
在WebGL中，顶点属性是作为顶点着色器的输入存在的。而在Oak3D中，网格属性是指顶点数组对象（Vertex Array Buffer），其中可以包含一个或多个顶点着色器属性的源数据。

创建一个新的顶点属性列表。 okMesh对象中的属性与WebGL中的属性略有不同。
在WebGL中，顶点属性是作为顶点着色器的输入存在的。而在Oak3D中，网格属性是指顶点数组对象（Vertex Array Buffer），其中可以包含一个或多个顶点着色器属性的源数据。

{Enum} Usage Optional
指定顶点属性的用途。可选的值是[STATIC_DRAW, DYNAMIC_DRAW, STREAM_DRAW]；默认值是STATIC_DRAW。

Parameters:
{String} IdxName
绘制使用的索引列表名称。默认的索引列表名称是“Default”。
{Enum} Topology Optional
绘制的基本几何元素类型，可选值是[POINTS, LINES, LINE_STRIP, LINE_LOOP, TRIANGLES, TRIANGLE_STRIP, TRIANGLE_FAN]。
如果本参数被忽略，那么本方法将使用索引列表内置的几何元素类型。
{Number} Offset Optional
指定从索引列表的第几个元素开始绘制。默认值是0。
{Number} Count Optional
指定绘制多少个索引列表元素。默认值是整个索引列表的长度。
注意, 此数量应与DrawMode中指定的元素类型匹配, 如DrawMode为OAK.TRIANGLES(即三角形列表), 则绘制索引个数应为3的倍数。

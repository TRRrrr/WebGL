{Enum} collideAABBox(目标包围盒。)
用此轴对齐包围盒与参数中给定的另一轴对齐包围盒做碰撞检测，并返回碰撞结果。
Parameters:
{okAABBox} 目标包围盒。
Returns:
{Enum} OAK.COLLIDE_TRUE意味着两个轴对齐包围盒有碰撞；
OAK.COLLIDE_FALSE意味着两个轴对齐包围盒没有碰撞。
{Enum} collideLine(LinePos0, LinePos1, Ret)
轴对齐包围盒与直线做碰撞检测。若有碰撞，则RetT.fT为碰撞点在直线上的系数。
Parameters:
{okVec3} LinePos0
直线的一个点。
{okVec3} LinePos1
直线的另一个点。
{Object} Ret
用于返回碰撞结果的容器，碰撞点是由直线的线性插值参数决定的。
InterPos = LinePos0 + Ret.fT * (LinePos1 - LinePos0).
Returns:
{Enum} OAK.COLLIDE_TRUE意味着直线和轴对齐包围盒有碰撞；
OAK.COLLIDE_FALSE意味着直线和轴对齐包围盒没有碰撞。
{Enum} collideRay(RayOrigin, RayDir, Ret)
轴对齐包围盒与射线做碰撞检测，若有碰撞，则RetT.fT为碰撞点在射线上的系数。
Parameters:
{okVec3} RayOrigin
射线的起点。
{okVec3} RayDir
射线的朝向。
{Object} Ret
用于返回碰撞结果的容器，碰撞点是由射线的线性插值参数决定的。
InterPos = RayOrigin + Ret.fT * RayDir.
Returns:
{Enum} OAK.COLLIDE_TRUE意味着射线和轴对齐包围盒有碰撞；
OAK.COLLIDE_FALSE意味着射线和轴对齐包围盒没有碰撞。

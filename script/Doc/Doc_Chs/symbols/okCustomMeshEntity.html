Class okCustomMeshEntity


Extends okVisibleEntity.
okCustomEntity用于渲染自定义几何体数据。

Class Summary
Constructor Attributes Constructor Name and Description
 
okCustomMeshEntity(zone)
okCustomEntity是okVisibleEntity的子类，允许开发者直接控制几何体数据。

createAttribute(Name, Len, Data, Usage)
创建一个新的顶点属性列表。 这里的属性与WebGL中的属性有些不同。在WebGL中，顶点属性是作为顶点着色器的输入存在的。而在Oak3D中，一个网格的属性是指Vertex Array Buffer，其中可以包含一个或多个顶点着色器的属性的数据源。

{Null} createAttribute(Name, Len, Data, Usage)
创建一个新的顶点属性列表。 这里的属性与WebGL中的属性有些不同。在WebGL中，顶点属性是作为顶点着色器的输入存在的。而在Oak3D中，一个网格的属性是指Vertex Array Buffer，其中可以包含一个或多个顶点着色器的属性的数据源。
Parameters:
{String} Name
顶点属性名。
{Number} Len
此属性流的长度（单位: 基本数据元素个数）。
{Array|TypedArray} Data Optional
用于初始化buffer的额数据。如果本参数被忽略，那么将初始化一个空的属性列表。
{Enum} Usage Optional
指定顶点属性的用途。可选的值是[STATIC_DRAW, DYNAMIC_DRAW, STREAM_DRAW]。默认值是STATIC_DRAW。

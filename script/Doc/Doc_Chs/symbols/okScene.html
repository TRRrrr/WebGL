Class okScene

Oak3D Engine的场景对象。

Class Summary
Constructor Attributes Constructor Name and Description
<private> <inner>  
okScene(sName, engine)
在Oak3D Engine中，场景是所有3D物体的容器，开发者可通过创建和操纵okScene对象来配置想要的3D场景内容。场景对象只能由okSceneManager#createScene方法创建。

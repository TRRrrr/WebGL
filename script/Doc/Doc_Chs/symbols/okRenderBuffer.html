Class okRenderBuffer

Render Buffer对象类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okRenderBuffer(Context)
Render Buffer是一种数据储存容器，其中保存了用于内部渲染的图像格式。Render Buffer一般作为独立的Depth/Stencil Buffer用于对Frame Buffer的渲染.
Method Summary
Method Attributes Method Name and Description
 
createBuffer(Format, Width, Height)
初始化一个指定格式和尺寸的WebGL Frame Buffer对象。
 
releaseBuffer()
释放当前的WebGL Render Buffer对象。 本方法将会释放此buffer中的WebGL资源。调用本方法后，此buffer将不再可用直至下次重新初始化。

{Boolean} loadModel(ModelName, ModelDoc, MeshMask, KeepMtrl, bKeepMtrl)
从外部加载一个模型文件。物件将根据ModelName给的相对地址，从场景的基准模型资源引用地址中去寻找这个模型资源。
在这里需要注意的是，即使直接提供模型资源的档案内容，在这里仍然需要给出正确的模型资源的路径名字，因为资源管理器以资源名作为引用关键词，所以正确的资源路径名称有利于引擎内部对资源的复用。 okDynamicEntity的几何体数据来源于模型资源。
Parameters:
{String} ModelName
模型资源的名称，即模型文件在场景模型资源引用地址下的相对地址或资源关键字。
{String} ModelDoc Optional
模型资源文档的内容。如果本参数没有被忽略，那么本方法将直接从此字符串中读取模型资源。
{Array} MeshMask Optional
本参数是一个包含若干Mesh名称的数组，如果设置了本参数，那么本方法将只会载入参数中指定的Mesh名称的mesh。
{Boolean} KeepMtrl Optional
保留参数
bKeepMtrl
Returns:
{Boolean} 如果载入成功，则返回true；反之，则返回false。

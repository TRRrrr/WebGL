Class okArrayBuffer

数组缓冲类。

Class Summary
Constructor Attributes Constructor Name and Description
 
okArrayBuffer(Context)
okArrayBuffer是WebGL标准中Array Buffer在Oak3D中的对象封装，CPU端可以分配，初始化一段存储器区域，并绘制这段存储器区域中的数据。
WebGL标准中定义, 使用WebGL接口进行3D图形渲染必须将顶点/索引等几何体描述数据储存于Array Buffer对象，Array Buffer对象是由WebGL托管的渲染数据容器, WebGL会根据开发者指定的Array Buffer用途将数据放在最合适的储存空间中（内存、显存或AGP存储器）。

{Number} Stride Optional
指定相邻两顶点距离的元素数, 一般用于Array Buffer存储多类数据的情况,。
比如：Array Buffer存储顶点位置与法线, 形式为<x, y, z> <nx, ny, nz> <x, y, z>……; 则位置Attribute的stride应为6.

{Null} createBuffer(BufType, DataType, Usage, DataPara)
初始化Array Buufer对象。 本方法用于初始化Array Buffer对象。
开发者可以使用指定的数据初始化一个Array Buffer对象，也可以创建一个固定长度的空的Array Buffer对象。
Parameters:
{Enum} BufType
指定Array Buffer类型。
作为顶点数据，应指定为ARRAY_BUFFER；作为索引数据，应指定为ELEMENT_ARRAY_BUFFER。
{Enum} DataType
指定Array Buffer的数据类型。
作为顶点数据，应指定为FLOAT。
作为索引数据, 应指定UNSIGNED_SHORT。
{Enum} Usage Optional
指定Array Buffer的用途。
STATIC_DRAW表明当前buffer仅初始化一次, 频繁用于渲染。
DYNAMIC_DRAW表明当前buffer会频繁更新, 并频繁用于渲染；
STREAM_DRAW表明当前buffer更新一次, 但并不会频繁用于渲染；
默认值是STATIC_DRAW。

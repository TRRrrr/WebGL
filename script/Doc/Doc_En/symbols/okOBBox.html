Class Detail
okOBBox(MinPos, MaxPos, Mat)
The arbitrarily oriented minimum bounding box is the minimum bounding box, calculated subject to no constraints as to the orientation of the result.
Parameters:
{okVec3} MinPos Optional
The minimal point of the OBB in local space. The default value is <0, 0, 0>.
{okVec3} MaxPos Optional
The maximal point of the OBB in local space. The default value is <0, 0, 0>.
{okMat43} Mat Optional
The transform matrix. The default value is a identity matrix.
Returns:
{okOBBox} A new okOBBox object
Field Detail
{okMat43} mMat
The transform matrix of the OBB. This matrix will transform OBB from its local space to world space. Actually, we use a AABB to represent the OBB in local space.
{okVec3} vMax
The maximal point of the OBB in local space.
{okVec3} vMin
The minimal point of the OBB in local space.

{Enum} collideLine(LinePos0, LinePos1, Ret)
Check if a line is intersected with the OBB and calculate the intersecting point.
Parameters:
{okVec3} LinePos0
A position to define the line.
{okVec3} LinePos1
A position to define the line.
{Object} Ret
This is a container to return the intersecting point defined using the linear interpolation parameter of the line. InterPos = LinePos0 + Ret.fT * (LinePos1 - LinePos0).
Returns:
{Enum} OAK.COLLIDE_TRUE means the line has intersection with the OBB. OAK.COLLIDE_FALSE means the line doens't have intersection with the OBB.
{Enum} collideRay(RayOrigin, RayDir, Ret)

Parameters:

{okVec3} RayDir
{Object} Ret
This is a container to return the nearest intersecting point defined using the linear interpolation parameter of the ray. InterPos = RayOrigin + Ret.fT * RayDir.
Returns:
{Enum} OAK.COLLIDE_TRUE means the ray has intersection with the OBB. OAK.COLLIDE_FALSE means the ray doesn't have intersection with the OBB.

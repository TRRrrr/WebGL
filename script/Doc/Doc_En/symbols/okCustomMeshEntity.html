{Null} createAttribute(Name, Len, Data, Usage)
Create a attribute array. The attribute is a little different from WebGL. In WebGL, a vertex attribute is a vertex shader input. For Oak3D, a mesh attribute is a vertex array buffer, it can contain the data source for one or more vertex shader attributes.
Parameters:
{String} Name
Attribute name.
{Number} Len
Length of the attribute array(counted by base element).
{Array|TypedArray} Data Optional
The data content used to initialize the buffer. If this parameter is ignored, then the new attribute array buffer will be initialized to empty.
{Enum} Usage Optional
The usage of this attribute array buffer. The available enumeration is [STATIC_DRAW, DYNAMIC_DRAW, STREAM_DRAW]. The default value is STATIC_DRAW.

{Null}

{okMesh} getMesh()
Get the built-in okMesh object. The geometry data of the okCustomMeshEntity is kept in a okMesh object. Commonly, the built-in okMesh object should be managed by entity instead of the user, but if you need to operate the mesh directly, then you can get the okMesh object by this method. For example, if you want to create a sphere mesh using the Oak3D utility: okGenSphereMesh(entity.getMesh(), 1, 16, 16);

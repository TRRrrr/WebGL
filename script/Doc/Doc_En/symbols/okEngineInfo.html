Class Detail
okEngineInfo(rc)
Construct a new engine info object. It's to be used for querying platform related information and webgl capability. This object can not be created directly by the user. It's a built-in component of the okEngine object.
Parameters:
rc
Returns:
{okEngineInfo} A new system info object

Class Detail
okShader(Context, Type)
Shader allows applications to describe operations that occur on vertex/fragment values and their associated data by using a vertex/fragment shader.
Parameters:
{WebGLContext} Context
The webgl context of the target canvas.
{Enum} Type
The available enumeration is [VERTEX_SHADER, FRAGMENT_SHADER].
Returns:
{okShader} A new okShader object
Method Detail
{Boolean} compile(Def, RetErr)
Compile the shader. This method compiles the source code to the binary code which can be accepted by WebGL.
Parameters:
{Object} Def Optional
A dictionary of macro definitions while compiling.
{Object} RetErr Optional
A error information container. If shader compilation is failed, the error information will be stored in RetErr.errorMsg.
Returns:
{Boolean} True if the compilation is successful, otherwise, return false.

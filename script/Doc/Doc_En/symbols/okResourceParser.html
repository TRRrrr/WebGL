Parsing model resource document and load it to the specified okModel object.
Parameters:
{Enum} DocType
The format of the document. The available enumeration is [RESDOCTYPE_XML].
{okModel} Model
The model object to be used to contain the parsing result.

The raw string of the model resource document.

If this parameter is set, then only the meshes whose name are in this array will be loaded.

Indicates if generate a specific index array for wireframe rendering.

<static> {Boolean} okResourceParser.registerModelParser(DocType, Parser)
Register a custom model parser. Oak3D allows developers write their own custom model parsers to support various model formats.
Parameters:
{Enum} DocType
The format of the document. [RESDOCTYPE_CUSTOM + i (i >= 0)].
{Function} Parser
The parser function. The prototype of the parser function should be: Boolean function(okModel model, String docText, Array meshMask). This function should parses the input docText and fill the model info into the model object. The parameter 'meshMask' is an optional parameter to implement partly loading, if your parser doesn't need implement partly loading, just ignore it. The return value of the parser function indicates whether the parsing is successfully completed.
Returns:
{Boolean} True if the custom parser is successfully registered, otherwise, return false.
See:
okModel

{Null} setTexture(TexName, AlphaBit, Width, Height, FrameCount)
Set the texture of the particle sprite. The texture will be used as both the color map & the alpha map. And the texture can contain multiple sub images(frames) to implement animating texture effect. The frames in a animating texture should be positioned in row-major order.
Parameters:
{String} TexName
The name of the texture resource. This an be a url relative to the base texture url of the scene or a resource key.
{String} AlphaBit
Indicates which channel of the texture is used for alpha. The available candidates are ["r", "g", "b", "a"].
{Number} Width Optional
The width of one frame within the texture. This parameter can be ignored if no animating texture is needed.
{Number} Height Optional
The height of one frame within the texture. This parameter can be ignored if no animating texture is needed.
{Number} FrameCount Optional
The total count of the frames within this texture. This parameter can be ignored if no animating texture is needed.

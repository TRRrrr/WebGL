{Number} getShadowFade()
Get the shadow fade factor of this entity. Shadow fade factor is a coefficient for fading the shadow into the dark side of the object. A proper value of this factor will remove the artifacts caused by shadow map technique. The final fade value will be the max of the fade value of this entity and the fade value of the light.

okFrameBuffer(Context)
Framebuffer object encapsulates state necessary to describe a collection of color, depth, and stencil logical buffers. Textures or renderbuffers can be attached to the framebuffer to store the rendered output.
Parameters:
{WebGLContext} Context
The webgl context of the target canvas.
Returns:
{okFrameBuffer} A new okFrameBuffer object

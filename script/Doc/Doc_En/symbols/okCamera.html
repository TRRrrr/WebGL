okCamera(Context)
okCamera is a utility to help users controlling camera in 3D space. A camera in Oak3D is a collaboration of view point, projection mode and viewport.
Parameters:
{WebGLContext} Context
The webgl context of the target canvas.
Returns:
{okCamera} A new okCamera object

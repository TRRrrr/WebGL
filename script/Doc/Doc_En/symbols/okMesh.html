okMesh(Context)
Mesh is the standard geometry data container in Oak3D. okMesh object can be created on the fly or created within a okModel object.
Parameters:
{WebGLContext} Context
The webgl context of the target canvas.
Returns:
{okMesh} A new okMesh object

{Null} createAttribute(Name, Len, Data, Usage)
Create a attribute array. The attribute in okMesh is a little different from WebGL. In WebGL, a vertex attribute is a vertex shader input. For Oak3D, a mesh attribute is a vertex array buffer, it can contain the data source for one or more vertex shader attributes.
Parameters:
{String} Name
Attribute name.
{Number} Len
Length of the attribute array(counted by base element).
{Array|TypedArray} Data Optional
The data content used to initialize the buffer. If this parameter is ignored, then the new attribute array buffer will be initialized to empty.
{Enum} Usage Optional
The usage of this attribute array buffer. The available enumeration is [STATIC_DRAW, DYNAMIC_DRAW, STREAM_DRAW]. The default value is STATIC_DRAW.
Returns:
{Null}
{Null} createIndex(Name, Len, Data, Usage, Topology)

<private> <inner> okRenderer(engineSetting, engine)
The rendering operation and configuration can be performed through the renderer object. Renderer can not be created directly by the user. It's a built-in component of the okEngine object.
Parameters:
engineSetting
engine
See:
okEngine#getRenderer

{Null} beginView(Cam, KeepBackGround)
Begin calling rendering operations for a specified camera view. All of the rendering operations must be called between okRenderer#beginView and okRenderer#endView.
Parameters:
{okCamera} Cam
The camera to define the view.
{Boolean} KeepBackGround
Indicates if keeping the current background of the view. By default, this method will clear the view area with the back color of the camera. The default value is false.

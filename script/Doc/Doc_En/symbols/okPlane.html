{Enum} collideLine(LinePos0, LinePos1, Ret)
Check if a line is intersected with the plane and calculate the intersecting point.
Parameters:
{okVec3} LinePos0
A position to define the line.
{okVec3} LinePos1
A position to define the line.
{Object} Ret
This is a container to return the intersecting point defined using the linear interpolation parameter of the line. InterPos = LineOrigin + Ret.fT * LineDir.
Returns:
{Enum} OAK.COLLIDE_TRUE means the line has intersection with the plane. OAK.COLLIDE_FALSE means the line is parallel with the plane. OAK.COLLIDE_CONTAIN means the line is on the plane.

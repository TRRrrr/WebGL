okProgram(Context)
A program object is an object to which shader objects can be attached. This provides a mechanism to specify the shader objects that will be linked to create a program.
Parameters:
{WebGLContext} Context
The webgl context of the target canvas.
Returns:
{okProgram} A new okProgram object

{Boolean} link(RetErr)
Link this program. This method links the program object. Attached vertex shader objects are used to create an executable that will run on the programmable vertex processor. Attached fragment shader objects are used to create an executable that will run on the programmable fragment processor.
Parameters:
{Object} RetErr Optional
A error information container. If link operation is failed, the error information will be stored in RetErr.errorMsg.
Returns:
{Boolean} True if the link operation is successful, otherwise, return false.
{Null} releaseProgram()
Release the program. This method will release the WebGL resource generated within this program. After calling this method the program will become invalid until the next link operation.

{Null} setAttribute(AttribName, Buf, ElementCount, Offset, Stride)
Set a vertex attribute parameter.
Parameters:
{String} AttribName
The name of the vertex attribute paramter.
{okArrayBuffer} Buf
The array buffer.
{Number} ElementCount
Indicate a vertex needs how many base data elements. For example, if the data content of the buffer is used as vertex position <x,y,z>, then the position of one vertex is composed by 3 float data element.
{Number} Offset Optional
Specifies the element offset to the first component of the first vertex attribute in the array. The default value is 0.
{Number} Stride Optional
Specifies the element offset between consecutive vertex attributes. If stride is 0, the generic vertex attributes are understood to be tightly packed in the array. The default value is 0.
Returns:
{Null}

{Null} setUniformMat43(ParaName, Val, ConvertToMat4)
Set 4x3 matrix to a uniform parameter with mat4 type or 3 vec4 parameters. GLSL shader doesn't have base type for 4x3 matrix, but this method can convert a 4x3 matrix to 4x4 matrix or 3 quadvectors.
Parameters:
{String} ParaName
The name of the uniform paramter.
{okMat43} Val
The setting value.
{Boolean} ConvertToMat4
If this parameter is set to true, then the 4x3 matrix will be set to a uniform paramter with mat4 type. Otherwise, the 4x3 matrix will be splited to 3 vec4 rows and set to a vector array.
Returns:
{Null}

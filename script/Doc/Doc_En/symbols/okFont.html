Class okFont

Font represent a font with specified font attributes including fontfamily, fontsize,fontweight etc. All font-related operation interface are in this class.

Method Detail
drawText(targetRect, text, color, renderTarget)
Draw text to given area of render target with specified color and alpha.
Parameters:
{okRect} targetRect
The drawing area in renderTarget, if null, the area will be the whole render target.
{String} text
The text to draw.
{okVec4} color
The text color and alpha, in RGBA format, r, g, b, a is in [0, 1].
{okTexture} renderTarget
The render Target. If null, the text will be drawed to current font's render context.
enableAutoWrap(bAutoWrap)
Set whether the text will be auto wrapped when drawText function is called. Default is true.
Parameters:
{Boolean} bAutoWrap
true or false, whether the text will be wrapped when drawText function is called.
{Boolean} isAutoWrapEnabled()
Check whether the font is under auto wrap mode.
Returns:
{Boolean} True if this font object is under autowrap mode, otherwise, false.
setCodePageMaxNum(maxNum)
Set maximum codepage number allowed simultaneously for storing characters. This can affect the text module's rendering performance.
Parameters:
{Number} maxNum
The new max value of font codepage underlying, default is 5.
setCodePageSize(width, height)
Set the underlying size of a codepage. The code page's width and height value affect the number of different characters which can be stored in one codepage.
Parameters:
{Number} width
The new width value of underlying codepage, should in power of 2, if not, the function will automatically increase it to min power of 2. Default is 512.
{Number} height
The new height value of underlying codepage, should in power of 2, if not, the function will automatically increase it to min power of 2. Default is 512.
